
# Music Platform (Spotify-Style Fullstack App)

Frontend: HTML/CSS/JS (Cloudflare Pages)
Backend: Node.js + Express (Render)
Database: PostgreSQL (Aiven)

## Setup
1. Configure server/.env
2. Ensure Aiven DB already has schema + seed
3. Run backend:
   cd server
   npm install
   npm run dev

4. Open client/index.html with Live Server
